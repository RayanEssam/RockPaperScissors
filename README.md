# RockPaperScissors

<h3>  Downloading Steps  </h3>
to try the game all you have to do is : 

1. download the repo
2. open the project using Xcode 
3. run

<h3>  How to play?  </h3>

1. open the game 
2. choose an object (rock, paper, scissors)
3. click on the play button
4. the pop up alert will announce the result.
